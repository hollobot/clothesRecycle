# clothesRecycle-admin

校园二手衣物回收系统管理后台（Vue3 + Pinia + Vue Router + Axios）。

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 环境变量

`.env.development`

```env
VITE_API_BASE_URL=http://localhost:8080
```

## UI 说明

- 后台视觉风格与根目录 `prototype.html` 的绿色体系保持一致。
- 已提供管理框架与核心页面骨架（看板、校区、管理员、回收点、物品审核、用户、礼品）。
